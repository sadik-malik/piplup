# Installation

Run one of the following commands to add @piplup/rhf-core to your project:

```bash
npm install @piplup/rhf-core react-hook-form

# or

pnpm add @piplup/rhf-core react-hook-form

# or

yarn add @piplup/rhf-core react-hook-form
```

# Peer dependencies

Please note that react and react-hook-form are required peer dependencies, meaning you should ensure they are installed as well.

```json
{
  "react": ">=18.0.0",
  "react-hook-form": ">=7.51.0 <8.0.0"
}
```
